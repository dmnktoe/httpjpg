# Opacity

Transparency scale from fully transparent (`0`) to fully opaque (`100`). Use for disabled
states, muted text, and overlay scrims. The checkerboard behind each swatch reveals the
transparency.

## Usage

```tsx
import { Box } from "@httpjpg/ui";

<Box css={{ opacity: 50 }}>Half-transparent</Box>;
<Box css={{ opacity: 70 }}>Muted surface</Box>;
```

## Guidelines

- **Muted text**: `70`–`80`. **Disabled**: `50`. **Scrims/overlays**: `40`–`60`.
- Prefer a semantic color token (e.g. `pageMuted`) over stacking opacity on text where possible.

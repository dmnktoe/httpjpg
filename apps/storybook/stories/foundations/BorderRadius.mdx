# Border Radius

Corner-rounding scale, from sharp (`none`) to fully round (`full`). Brutalist surfaces stay
square — rounding is reserved for pills, avatars, and soft accents.

## Usage

```tsx
import { Box } from "@httpjpg/ui";

<Box css={{ borderRadius: "lg" }}>Rounded card</Box>;
<Box css={{ borderRadius: "full" }}>Pill / avatar</Box>;
```

## Guidelines

- **Structure stays square.** Use `none` for brutalist sections, cards, and dividers.
- **`full`** is only for circular avatars and pill buttons.
- Pick a single radius per surface — don't mix `sm` and `xl` on the same element.

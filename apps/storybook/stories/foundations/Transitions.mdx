# Transitions

Shared durations and easing curves. Keep motion short and consistent — most interactions use
`normal` (0.2s) with `easeInOut`. The dots below loop continuously so you can compare them.

## Usage

```tsx
import { Box } from "@httpjpg/ui";

<Box
  css={{
    transition: "all 0.2s ease-in-out",
    _hover: { transform: "translateY(-2px)" },
  }}
>
  Lifts on hover
</Box>;
```

## Guidelines

- **fast (0.1s)** — micro-feedback like color or opacity changes.
- **normal (0.2s)** — the default for most hover and state transitions.
- **slow / slower** — larger movements, panels, and entrances.
- Respect reduced motion: gate non-essential animation behind `motionSafe`.

## ðŸš€ Quick Start

Get started in 3 steps:

1. **Setup Environment Variables**
   ```bash
   STORYBLOK_MANAGEMENT_TOKEN=your-token
   STORYBLOK_SPACE_ID=your-space-id
   ```

2. **Sync Everything**
   ```bash
   pnpm --filter @httpjpg/storyblok-sync sync:all
   ```

3. **Start Developing**
   ```bash
   pnpm dev
   ```

# Typography

Strong typography hierarchy following brutalistic design principles.

## Font Families

### Sans Serif (Body Text)

Default font family for body text and UI elements.

**Stack**: Arial, Helvetica, sans-serif

<div style={{ fontFamily: 'Arial, Helvetica, sans-serif', fontSize: '1rem', marginBottom: '2rem' }}>
  The quick brown fox jumps over the lazy dog. 0123456789
</div>

### Headline (Impact)

Bold, impactful font for headlines and large text.

**Stack**: Impact, Haettenschweiler, Arial Narrow Bold, sans-serif

<div style={{ fontFamily: 'Impact, Haettenschweiler, Arial Narrow Bold, sans-serif', fontSize: '2rem', textTransform: 'uppercase', marginBottom: '2rem' }}>
  THE QUICK BROWN FOX
</div>

### Accent (Decorative)

Elegant serif for special moments and decorative text.

**Stack**: Trattatello, serif

<div style={{ fontFamily: 'Trattatello, serif', fontSize: '1.5rem', marginBottom: '2rem' }}>
  The quick brown fox jumps over the lazy dog
</div>

### Monospace (Technical)

Fixed-width font for code, technical details, and data.

**Stack**: ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, monospace

<div style={{ fontFamily: 'ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, monospace', fontSize: '0.875rem', marginBottom: '2rem' }}>
  The quick brown fox jumps over the lazy dog. 0123456789
</div>

### Size Scale

- **xs**: 0.75rem (12px) - Small text, captions
- **sm**: 0.875rem (14px) - Secondary text
- **base**: 1rem (16px) - Body text (default)
- **lg**: 1.125rem (18px) - Emphasized body text
- **xl**: 1.25rem (20px) - Small headings
- **2xl**: 1.5rem (24px) - Section headings
- **3xl**: 1.875rem (30px) - Page headings
- **4xl**: 2.25rem (36px) - Large headings
- **5xl**: 3rem (48px) - Hero headings
- **6xl**: 3.75rem (60px) - Display text
- **7xl**: 4.5rem (72px) - Large display
- **8xl**: 6rem (96px) - Extra large display
- **9xl**: 8rem (128px) - Maximum impact

## Usage Examples

### Headlines

```tsx
import { Headline } from '@httpjpg/ui';

<Headline level={1}>BRUTALIST DESIGN</Headline>
<Headline level={2}>Section Title</Headline>
<Headline level={3}>Subsection</Headline>
```

### Body Text

```tsx
import { Paragraph } from '@httpjpg/ui';

// Default (sm = 12px)
<Paragraph>
  Body text with optimal readability
</Paragraph>

// Medium (16px)
<Paragraph size="md">
  Emphasized body text
</Paragraph>

// Large (18px)
<Paragraph size="lg">
  Lead paragraph or important text
</Paragraph>
```

### Technical Text

```tsx
import { Box } from '@httpjpg/ui';

<Box css={{ fontFamily: 'mono', fontSize: 'xs' }}>
  2024-03-15 | Status: Active
</Box>
```

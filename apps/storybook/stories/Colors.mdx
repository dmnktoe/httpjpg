## Neutral Scale

Complete gray scale for backgrounds, borders, and text. Perfect for brutalist layouts that need strong contrast.

<ColorPalette>
  <ColorItem
    title="neutral"
    subtitle="Gray scale from light to dark"
    colors={Object.keys(colors.neutral).reduce((acc, key) => {
      acc[key] = colors.neutral[key];
      return acc;
    }, {})}
  />
</ColorPalette>

### Usage

```tsx
import { Box, Paragraph } from '@httpjpg/ui';

<Box css={{ bg: 'neutral.100', p: '24px' }}>
  <Paragraph css={{ color: 'neutral.700' }}>
    Body text on subtle background
  </Paragraph>
</Box>
```

## Color Semantics

### Text Colors

- `black` - Primary text (default)
- `neutral.700` - Body text
- `neutral.500` - Secondary text
- `neutral.400` - Disabled text
- `white` - Inverted text
- `primary.500` - Primary links and actions
- `secondary.500` - Secondary emphasis
- `accent.500` - Highlights, creative emphasis

### Background Colors

- `white` - Default background
- `neutral.50` - Subtle background
- `neutral.100` - Section background
- `black` - Dramatic sections
- `primary.50` - Subtle brand background
- `secondary.50` - Subtle secondary background
- `accent.50` - Subtle accent background

### Border Colors

- `neutral.200` - Default borders
- `neutral.300` - Hover borders
- `neutral.400` - Active borders
- `black` - Strong borders
- `primary.500` - Brand borders
- `secondary.500` - Secondary borders
- `accent.500` - Accent borders

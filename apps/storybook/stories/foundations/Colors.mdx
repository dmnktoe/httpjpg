# Colors

A monochromatic foundation — black, white, and a full neutral scale — paired with two
vibrant brand colors and a set of semantic status colors. Brutalist by default: strong
contrast, sparing accents.

## Semantic Tokens

Theme-aware tokens that resolve to different palette values in light and dark mode. Prefer
these over raw scales for any surface that should respond to the theme.

- `pageBg` — page background (white → black)
- `pageFg` — page foreground / text (black → white)
- `pageMuted` — muted text (`neutral.700` → `neutral.300`)
- `pageBorder` — default border (`neutral.300` → `neutral.700`)

## Usage

```tsx
import { Box, Button, Paragraph } from "@httpjpg/ui";

// Theme-aware surface
<Box css={{ bg: "pageBg", color: "pageFg", p: "24px" }}>
  <Paragraph css={{ color: "pageMuted" }}>Body text that follows the theme</Paragraph>
</Box>;

// Brand actions
<Button variant="primary">Primary Action</Button>;
<Button variant="accent">Accent Action</Button>;
```

### Quick reference

| Role               | Token                                         |
| ------------------ | --------------------------------------------- |
| Primary action     | `primary.500`                                 |
| Accent / highlight | `accent.500`                                  |
| Destructive        | `danger.500`                                  |
| Low-emphasis       | `neutral.200` – `neutral.400`                 |
| Text               | `pageFg`, `pageMuted`, `neutral.500`          |
| Surfaces           | `pageBg`, `neutral.50`, `neutral.100`         |
| Borders            | `pageBorder`, `neutral.300`                   |

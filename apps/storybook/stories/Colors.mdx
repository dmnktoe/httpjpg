## Color Semantics

### Text Colors

- `black` - Primary text (default)
- `neutral.700` - Body text
- `neutral.500` - Secondary text
- `neutral.400` - Disabled text
- `white` - Inverted text
- `primary.500` - Links, primary actions
- `accent.500` - Highlights, emphasis

### Background Colors

- `white` - Default background
- `neutral.50` - Subtle background
- `neutral.100` - Section background
- `black` - Dramatic sections
- `primary.50` - Subtle brand background
- `accent.50` - Subtle accent background

### Border Colors

- `neutral.200` - Default borders
- `neutral.300` - Hover borders
- `neutral.400` - Active borders
- `black` - Strong borders
- `primary.500` - Brand borders
- `accent.500` - Accent borders
